## Overview
This project implements a singular-brain robot swarm for optimized path planning in highly dynamic environments. Built with ROS and Gazebo, the system enables multiple robots to collaboratively explore and navigate a workspace while adapting to moving obstacles and environmental changes. Users provide task instructions via NLP input, and the swarm performs efficient task allocation based on distance. It also features object detection and map merging using SLAM to maintain a shared, up-to-date representation of the environment. The system efficiently coordinates tasks and scales to larger workspaces and robot teams.
